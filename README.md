# SnapFoodBackend

-------------------------------
SnappFood Backend with JavaScript and Express.js
------------------------------
Tech Stack

Server: Node.js, Express

DataBase: MongoDB
-------------------------------
 dependencies

    bcrypt
    dotenv
    express
    jsonwebtoken
    mongoose
    multer
    nodemailer
    yup
    nodemon
-------------------------------
Run Locally

Clone the project

  git clone git@github.com:afotohi12/SnapFoodDelivery.git
  
Go to the project directory

  cd SnapFoodDelivery
Install dependencies

  npm install
Start the server

  npm start
